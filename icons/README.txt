
App Icon Package
================

This package contains your app icon in multiple sizes for different platforms:

iOS Folder:
- Contains all required sizes for iOS apps
- Use these for your Xcode project

Android Folder:
- Contains all required sizes for Android apps
- Use these for your Android Studio project

Web Folder:
- Contains sizes suitable for web applications and PWAs
- Use these for favicons and web app manifests

Original Folder:
- Contains the high-resolution original image
- Use this for any custom sizing needs
